Optimus
=======

_DOM Transcoding using Symfony2 components_

## Installation

```
$ git clone https://github.com/RobMasters/Optimus.git
$ cd Optimus

$ curl -s https://getcomposer.org/installer | php
$ php composer.phar install

```

## Testing

```
$ php composer.phar install --dev
$ vendor/bin/phpunit

```
